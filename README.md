## 🚀 Overview

A comprehensive fraud detection system that analyzes transactional banking data to identify suspicious behavioral patterns and potential fraudulent activities. The system combines machine learning, anomaly detection, and real-time risk scoring to provide actionable fraud intelligence.

## ✨ Features

- **Real-time Fraud Detection**: ML models (Random Forest, Isolation Forest, Logistic Regression)
- **Anomaly Detection Pipeline**: Unsupervised learning for unknown fraud patterns
- **Feature Engineering**: Time-based, rolling statistics, and behavioral features
- **Interactive Dashboards**: Power BI integration for real-time monitoring
- **Alert System**: Threshold-based fraud alerts
- **SQL Analytics**: Pre-built queries for fraud pattern analysis
- **Risk Scoring**: Real-time transaction risk assessment

## 📊 Performance Metrics

| Model | ROC-AUC | Precision | Recall | F1-Score |
|-------|---------|-----------|--------|----------|
| Random Forest | 0.94 | 0.89 | 0.76 | 0.82 |
| Isolation Forest | 0.87 | 0.71 | 0.68 | 0.69 |
| Logistic Regression | 0.91 | 0.82 | 0.72 | 0.77 |
